## 🚀 How To Use It

### Step 1: Setup (One Time)
```bash
cd cricket-trading-platform
npm install
```

### Step 2: Run It
```bash
# Option 1: Use script
./start.sh

# Option 2: Manual
npm start              # Backend (Terminal 1)
npm run frontend       # Frontend (Terminal 2)
```

### Step 3: Access
- **User App**: http://localhost:3000
- **Admin Panel**: http://localhost:3000/admin
- **API**: http://localhost:3001/api

### Step 4: Login
**Admin Account** (Pre-created):
- Email: `admin@crickettrading.com`
- Password: `admin123`

**User Account**:
- Click "Sign Up" and create one!
